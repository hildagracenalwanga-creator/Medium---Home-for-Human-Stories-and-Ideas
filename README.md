# Medium - Human Stories and Ideas  - Group 5b

A responsive Medium.com landing page clone built with HTML and CSS. Created as the capstone project for the Frontend Development course at BeTechified.

## Project Overview

This project replicates the core visual and functional elements of the Medium.com landing page. It demonstrates proficiency in semantic HTML5, modern CSS layout techniques, and responsive design principles.

The page features a fixed navigation bar with an interactive sign-in dropdown, a hero section with a call-to-action button, and a structured footer. The design is clean, minimal, and adapts seamlessly across desktop, tablet, and mobile screens.

## Features

Fixed navigation bar with Medium logo and navigation links
Interactive Sign in dropdown with multiple login options
Hero section with headline, subtitle, and call-to-action button
Responsive layout that works on all screen sizes
Mobile hamburger menu for smaller devices
Subtle hover effects on buttons and interactive elements
Semantic HTML5 structure for better accessibility

## How to Run

Clone the repository:
git clone https://github.com/hildagracenalwanga-creator/Medium---Home-for-Human-Stories-and-Ideas.git

Navigate to the project folder:
cd Medium---Home-for-Human-Stories-and-Ideas

Open index.html in your web browser.
Alternatively, use the Live Server extension in VS Code.
